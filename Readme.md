
# throttle(1)

 Tool for throttling JSON streams.

## Installation

```
$ go get github.com/segmentio/throttle
```

## Usage

```
  Usage:
    throttle <ops>
    throttle <ops> <duration>
    throttle -h | --help
    throttle --version

  Options:
    -h, --help        output help information
    -v, --version     output version
```
